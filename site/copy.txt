SECTION: NO SLOP. HERE'S THE MATH.
----------------------------------
H2: You've been flying blind.
Lead: Unlocking the black box token usage

pace    to_spend = week_left ÷ 5h_windows_left
        The sustainable token usage for this next 5hr period.
anchor  cap = burned ÷ used%
        Pinned to Claude's own /usage numbers every refresh.
burn    per line = (input + 5·output + 1.25·cache_write + 0.1·cache_read) × model_price
        Different models cost different
models  haiku ⅓ · sonnet 1 · opus 5⁄3 · fable 10⁄3
        Price-weighted. Refreshed daily from Anthropic's price sheet.

Footnote: Fable > Opus > Sonnet > Haiku, so your token usage should be pinned to market prices.

SECTION: INSTALL
----------------
H2: Install
Lead: One line in your terminal. Restart Claude Code. Done.
Hint 1: Prefer the plugin manager (skill only, no bar)? In Claude Code: /plugin marketplace add goodindustries/Maxx then /plugin install maxx@maxx.
Hint 2: Then type /maxx any time for total tokens, tokens/day, cache-hit rate and streak. Or /maxx session for how much to burn this window.
